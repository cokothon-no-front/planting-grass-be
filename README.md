# spring-webflux-usersave
개인 저장기능을 이용한 다용도 웹 서비스

## Used Library from Maven
- Swagger: [springfox-boot-starter](https://mvnrepository.com/artifact/io.springfox/springfox-boot-starter)
- JWT: [jjwt](https://mvnrepository.com/artifact/io.jsonwebtoken/jjwt-api)

## Tech Stack
- Kotlin
- SpringBoot
- R2dbc with JasyncSQL
- WebFlux
- Spring Reactive Security
