## 👊기획의도
- 태스크 관리를 보다 직관적으로.
- 친구들과 함께 오늘 해야 할 일을 오늘 해내기.
- 어제보다 더 나은 내가 되기 위해.
- 내 청춘을 더 푸르게, 잔디 심기
***

## 🔎프로젝트 소개
- 자체 로그인 기능을 이용해 고유 JSON Web Token 반환 로직 구현
- Data JPA를 이용한 MySQL DB관리
- 싱글 테이블 DB를 이용한 LIKE 쿼리문 사용
***

## ⚒️Tools
- postman을 이용해 API test

- Intellij IDEA를 이용해 프로젝트 빌드
